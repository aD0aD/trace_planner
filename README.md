# Trace Planner

Trace Planner 是一个面向徒步/越野路线的在线轨迹拼接与分析工具。  
无需安装客户端，直接在浏览器中完成导入、拼接、分析、标注与导出。

## Website

- [https://ad0ad.github.io/trace_planner/](https://ad0ad.github.io/trace_planner/)

## Documentation

- 中文手册: [README.zh.md](README.zh.md)
- English Guide: [README.en.md](README.en.md)

## What You Can Do

- Import `KML / KMZ / GPX` tracks
- Clip and stitch segments from multiple tracks
- Analyze distance, ascent, and descent by segment
- Add text/image annotations
- Export `KML / GPX / GeoJSON` and elevation chart PNG
- Save/load full project as `project.json`
- Switch UI language (中文 / English)
- Switch theme (Light / Dark)

## Privacy

All route/project files are processed locally in your browser.  
The website does not upload or store your files.
