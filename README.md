# Trace Planner · 在线轨迹拼接与分析

在浏览器里打开网页即可使用：**不用安装软件，不用把 HTML 下载到本地**。  
适合徒步、越野路线复盘：多轨迹导入、拼接、分段统计、标注、海拔图与导出。

**官方站点（示例）**：[https://aD0aD.github.io/trace_planner/](https://aD0aD.github.io/trace_planner/)  
（若你打不开，多半是网络或 Pages 未部署；维护者请看仓库内的 [`DEPLOY.md`](DEPLOY.md)。）

---

## 隐私说明（很重要）

- 轨迹、标注、工程文件 **只在你自己的浏览器里处理**。  
- **不会**上传到我们或任何第三方的服务器（本站为纯静态页面）。  
- 地图底图使用公开的在线瓦片服务，**会访问地图商的 CDN**（与打开普通地图网站相同）。

---

## 第一次怎么用？

1. 用电脑或手机浏览器打开上面的 **官方站点** 链接（推荐电脑，屏幕大更好操作）。  
2. 页面加载后进入主界面（若首页自动跳转，稍等即可）。  
3. 在 **「导入多条轨迹」** 里选择你的 `KML` / `KMZ` / `GPX` 文件。  
4. 需要拼接时看 **「轨迹拼接工具」** 标签；需要分析、导出、工程存盘看 **「轨迹分析工具」** 标签。

---

## 工程 / 项目：保存与再次打开（无需登录）

没有账号系统，数据由你本地掌控，有两种方式：

| 方式 | 适合场景 |
|------|----------|
| **下载工程 JSON** | 换电脑、长期备份、发给队友 |
| **保存到本浏览器** | 同一台电脑下次接着做（清缓存可能丢失） |

**推荐操作：**

1. 做完一段工作后点 **「下载工程 JSON」**，会得到 `xxx.project.json`。  
2. 下次在网页上点 **「打开工程 JSON」**，或把该文件 **拖到页面上的虚线框** 里。  
3. 首次打开网站时，若本机曾暂存过工程，可能会询问是否 **自动恢复**。

> 工程里包含：当前主轨迹、导入的多条轨迹、拼接篮子、断点、标注、爬升算法选项等。

---

## 轨迹拼接工具（怎么用）

1. **导入多条轨迹** 后，在列表里对某条点 **「设为分析轨迹」**，地图上黄色线为当前主轨迹。  
2. **片段截取**：拖动「起点 / 终点」滑块，或在地图上 **「地图选起终点」**（可反穿）。  
3. 看 **该段距离、爬升、下降**；下方 **实时高度图** 会随起终点移动。  
4. 点 **「加入拼接篮子」**，可多次从不同轨迹加段。  
5. 在篮子里 **上移 / 下移 / 反向 / 删除** 片段，确认顺序后点 **「生成拼接轨迹」**。  
6. 左侧图层控件可开关：卫星图、地形图、阴影、导入轨迹、断点等。

---

## 轨迹分析工具（怎么用）

1. **分析目标轨迹**：在顶部确认当前分析的是 **拼接结果** 还是某条 **导入轨迹**（下拉框切换）。  
2. **爬升计算选项**：可选「原始 / 平滑 / 平滑+阈值」，并可调平滑窗口与阈值；**不同设备导出的海拔噪声不同，可切换对比**。  
3. **断点分段**：用滑块选索引 → **添加断点**，或 **按段数自动断点**；表格里看每一段的距离与爬升下降；断点会在地图上显示。  
4. **路线标注**：选类型 → **添加标注** → 在地图上点击；文字标签可 **拖拽** 调整位置。  
5. **导出**：可导出 `KML` / `GPX` / `GeoJSON`，以及 **爬升下降图 PNG**。

---

## 支持的文件格式

**可导入：** `KML`、`KMZ`、`GPX`  

**可导出：** `KML`、`GPX`、`GeoJSON`、海拔图 `PNG`、工程 `project.json`

---

## 使用注意

- 需要 **联网**（地图与部分脚本从 CDN 加载）。  
- 轨迹点特别多时，老设备可能略卡。  
- **本浏览器暂存** 有容量限制；工程很大时请 **务必下载 JSON** 备份。
