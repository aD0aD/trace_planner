# Trace Planner（中文使用手册）

Trace Planner 是一个在线轨迹拼接与分析工具，适合徒步、越野、拉练路线复盘。  
直接在浏览器使用，无需安装、无需登录。

- 官方网站: [https://ad0ad.github.io/trace_planner/](https://ad0ad.github.io/trace_planner/)
- 支持语言: 中文 / English（页面右上角切换）
- 支持主题: 浅色 / 深色（页面右上角切换）

---

## 一、核心功能

### 1) 轨迹导入与显示
- 支持导入 `KML` / `KMZ` / `GPX`
- 支持同时导入多条轨迹
- 地图可切换卫星图 / 地形图 / 街道图，并可叠加阴影地形

### 2) 轨迹拼接
- 可在任意导入轨迹上选择起点终点截段
- 支持反向片段（反穿）
- 支持拼接篮子排序、删除、反向后生成新轨迹

### 3) 分析能力
- 总距离、总爬升、总下降统计
- 断点分段分析（每段距离/爬升/下降）
- 实时高度图联动起终点
- 爬升算法支持原始 / 平滑 / 平滑+阈值

### 4) 标注系统
- 支持不同颜色类型（起点、终点、营地、补给、风险、普通）
- 地图点击添加标注
- 标注支持文字 + 可选图片
- 标注列表支持：定位 / 编辑 / 删除

### 5) 导出与工程
- 导出轨迹：`KML` / `GPX` / `GeoJSON`
- 导出图表：爬升下降图 `PNG`
- 工程保存：下载 `project.json` 或保存到本浏览器

---

## 二、快速上手（推荐流程）

1. 打开网站，切换到你习惯的语言和主题  
2. 在“导入多条轨迹”中选择文件  
3. 在“轨迹拼接工具”里截段并加入拼接篮子  
4. 点“生成拼接轨迹”  
5. 在“轨迹分析工具”里查看分段统计、高度图、标注  
6. 导出轨迹或下载工程 JSON

---

## 三、工程保存说明（非常重要）

你有两种保存方式：

- **下载工程 JSON（推荐）**  
  - 可跨设备、可备份、可分享给队友
- **保存到本浏览器**  
  - 仅当前浏览器可见，清缓存或换设备会丢失

建议：每次重要编辑后，都下载一份 `project.json`。

## 五、隐私与数据安全

- 所有轨迹和工程文件都在你的浏览器本地处理
- 网站不会上传或保存你的文件
- 地图底图属于第三方在线瓦片服务（正常地图访问行为）

## 六、常见问题

### Q1: 为什么导入后只看到部分轨迹？
可能是原始文件中存在多段/重复点/异常点，建议先用单条文件测试，再逐步导入。

### Q2: 为什么不同文件的爬升差异很大？
海拔采样设备和噪声不同，建议切换不同爬升算法进行对比。

### Q3: 工程加载失败怎么办？
优先检查 JSON 是否被手动编辑或损坏；尽量使用工具导出的原始 `project.json`。
