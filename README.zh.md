# Trace Planner（中文使用手册）

在线轨迹拼接与分析工具，直接在浏览器使用，无需安装。

- 官方网站: [https://ad0ad.github.io/trace_planner/](https://ad0ad.github.io/trace_planner/)
- 支持语言: 中文 / English（网页右上角切换）

## 主要功能

- 导入多条轨迹：`KML` / `KMZ` / `GPX`
- 轨迹拼接：截段、反向、排序后生成新轨迹
- 断点分段统计：每段距离、爬升、下降
- 实时高度图：随起终点拖动联动
- 标注系统：文字标注 + 可选图片
- 导出：`KML` / `GPX` / `GeoJSON` / 爬升下降图 PNG
- 工程保存：下载 `project.json` 或保存到本浏览器

## 快速开始

1. 打开网站
2. 在“导入多条轨迹”中选择文件
3. 在“轨迹拼接工具 / 轨迹分析工具”完成操作
4. 导出成果或下载工程 JSON

## 工程保存建议

- **下载工程 JSON**：推荐，便于备份与跨设备
- **保存到本浏览器**：仅本机有效，清缓存可能丢失

## 隐私说明

- 轨迹与工程数据仅在浏览器本地处理
- 站点不上传、不存储你的文件
